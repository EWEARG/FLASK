La guía definitiva de Flexbox

El módulo Flex Layout es un nuevo módulo de diseño en CSS3 creado para mejorar la alineación, las direcciones y el orden de los elementos. en el contenedor incluso cuando son de tamaño dinámico o incluso desconocido. La característica primordial de la contenedor flexible es la capacidad de modificar el ancho o la altura de sus elementos secundarios para llenar el espacio disponible de la mejor manera posible en diferentes tamaños de pantalla.

Flexbox es relativamente nuevo, pero hoy en día disfruta de una excelente compatibilidad con los navegadores (casi el 83 % de los navegadores admiten caja flexible). Flexbox Layout se usa para componentes de aplicaciones pequeñas, mientras que CSS Grid Layout Module es destinado a manejar los diseños a gran escala.

Los dos ejes de Flexbox

Cuando trabaje con flexbox, debe pensar en términos de dos ejes: el eje principal y el eje transversal. El el eje principal está definido por la propiedad de dirección de flexión, y el eje transversal corre perpendicular a él. Todo lo que hacemos con flexbox se refiere a estos ejes, por lo que vale la pena entender cómo funcionan desde El principio.
